# DearLeaderBot
The tyrannical overlord of SocLit's discord server

# Running the project
1. clone the repo
2. download node.js
3. run `npm init -y`
4. run `npm install --save discord.js dotenv`
5. create a file called '.env' with the line `DISCORD_TOKEN=[authorization token]`
6. run `node bot.js`
7. enjoy!
